# Colors

Mainsail color classes are available.

## Background Colors

Apply Mainsail color to text `background-color:` css property.

```html
<div class="bg-blue-primary p-8">bg-blue-primary</div>
<div class="bg-blue-light p-8"></div>
<div class="bg-blue-middle p-8"></div>
<div class="bg-blue-dark p-8"></div>

<div class="bg-green-light p-8"></div>
<div class="bg-green-middle p-8"></div>
<div class="bg-green-dark p-8"></div>

<div class="bg-violet-light p-8"></div>
<div class="bg-violet-middle p-8"></div>
<div class="bg-violet-dark p-8"></div>

<div class="bg-orange-light p-8"></div>
<div class="bg-orange-middle p-8"></div>
<div class="bg-orange-dark p-8"></div>

<div class="bg-red-light p-8"></div>
<div class="bg-red-middle p-8"></div>
<div class="bg-red-dark p-8"></div>

<div class="bg-neutral-1 p-8"></div>
<div class="bg-neutral-2 p-8"></div>
<div class="bg-neutral-3 p-8"></div>
<div class="bg-neutral-4 p-8"></div>
<div class="bg-neutral-5 p-8"></div>
<div class="bg-neutral-6 p-8"></div>
```

## Text Colors

Apply Mainsail color to text `color:` css property.

```html
<div class="text-blue-primary p-8"></div>
<div class="text-blue-light p-8"></div>
<div class="text-blue-middle p-8"></div>
<div class="text-blue-dark p-8"></div>

<div class="text-green-light p-8"></div>
<div class="text-green-middle p-8"></div>
```
