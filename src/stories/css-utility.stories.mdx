# Utility Classes

Utility classes can offer some convenient access to Mainsail styling. They are inspired by [Tailwind CSS](https://tailwindcss.com).

## Table of Contents

-   [Spacing](#spacing)
-   [Shadows](#shadows)

### Spacing

For common Mainsail spec spacing margins and padding, you can use `p-<size>` and `m-<size>` classes.

These are controllable with an intuitive explicit sizing scale.

**Available Sizes**

```
0px,
4px,
8px,
10px,
12px,
15px,
20px,
30px,
48px,
```

Explicit directional margin/padding can be applied by adding a `t`/`b`/`l`/`r` designation to the class.

E.g. `mr-8` will apply `margin-right: 8px` and `ml-8` will apply `margin-left: 8px`.

If desired, you can supply `x` or `y` to apply to both sides.

**Examples**

```html
// Using margin utility with 8px of top margin
<button class="mainsail-button secondary mt-8">Cancel</button>

// Using margin utility with 8px of top and 20px of left/right margin
<button class="mainsail-button primary mt-8 mx-20">Confirm</button>
```

### Shadows

Box shadows can be applied easily with the shadow utility.

```html
<div class="px-15 py-8">Shadow Box</div>
```
