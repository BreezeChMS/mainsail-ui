# Spacing

For common Mainsail spec spacing margins and padding, you can use `p-<size>` and `m-<size>` classes.

These are controllable with an intuitive explicit sizing scale that comes from sizes used directly from our Figma specs.

:::warning
Note: These spacing utility classes utilize `!important` tags to ensure they can be used as overrides of existing styles.
:::

## Available Sizing Scale

| Class specifier | Pixels |
| --------------: | ------ |
|               0 | 0px    |
|               2 | 2px    |
|               4 | 4px    |
|               6 | 6px    |
|               8 | 8px    |
|              10 | 10px   |
|              12 | 12px   |
|              15 | 15px   |
|              16 | 16px   |
|              20 | 20px   |
|              30 | 30px   |
|              48 | 48px   |

## Margins

Explicit directional margin/padding can be applied by adding a `t`/`b`/`l`/`r` designation to the class.

E.g. `mr-8` will apply `margin-right: 8px` and `ml-8` will apply `margin-left: 8px`.

If desired, you can supply `x` or `y` to apply to both sides.

**Examples**

```jsx
// Using margin utility with 8px of top margin
<Button className="mt-8" text="Cancel" />

// Using margin utility with 8px of top and 20px of left/right margin
<Button className="mt-8 mx-20" text="Confirm"/>
```

## Space Between

If you need to quickly apply some in-between spacing in a simple way use, `space-y-<amount>` and `space-x-<amount>` (where amount is an integer in the table above) on a parent item. This adds margin to all but the first item of children.

If you have complex scenarios, reach for [AutoGrid](./?path=/docs/layout-grid-autogrid--simple-grid).

<div className="space-y-20 mb-20">
    <span className="body-text mb-20">
        This example uses <code>space-y-12</code> for automatic vertical
        spacing.
    </span>
    <div className="space-y-12 bg-red-light">
        <div className="bg-blue-light p-8"></div>
        <div className="bg-blue-light p-8"></div>
        <div className="bg-blue-light p-8"></div>
    </div>
</div>
<div className="mb-20 space-y-20">
    <span className="body-text">
        This example uses <code>space-x-12</code> for automatic horizontal
        spacing.
    </span>
    <div className="space-x-12 bg-red-light flex">
        <div className="bg-blue-light flex-grow p-8"></div>
        <div className="bg-blue-light flex-grow p-8"></div>
        <div className="bg-blue-light flex-grow p-8"></div>
    </div>
</div>

:::warning
No responsive features exist for the `space-` utility at this time. Beware of wrapping elements.
:::

## Responsive Spacing & Hiding

Responsive designations for spacing can be applied. The syntax looks like `<breakpoint>:<class>`

```jsx
<div className="bg-blue-light p-8 sm:mt-8 md:mt-20 lg:mt-30">mt-48</div>
```

<div>
    <span className="m-0">
        Margin with 8px at small, 20px at medium and 30 at large (resize browser
        window to see)
    </span>
    <div className="body-text bg-red-light">
        <div className="bg-neutral-1 p-4"></div>
        <div className="bg-blue-light p-20 sm:mt-8 md:mt-20 lg:mt-30">
            <span className="md:hidden lg:hidden">
                Small and under <code>mt-8</code>
            </span>
            <span className="sm:hidden lg:hidden">
                Medium breakpoints only <code>mt-20</code>
            </span>
            <span className="md:hidden sm:hidden">
                Large and up <code>mt-30</code>
            </span>
        </div>
    </div>
</div>
