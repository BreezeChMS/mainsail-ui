# Typography

Basic Mainsail typography classes are available.

<TOCInline toc={toc} />

## Headings

**Example**

```html
<h1>Hello World Hero</h1>
<h2>Hello World H2</h2>
<h3>Hello World H3</h3>
<h4>Hello World H4</h4>
```

<div className="bg-blue-light rounded p-20 mb-20">
    <h1>Hello World Hero</h1>
    <h2>Hello World H2</h2>
    <h3>Hello World H3</h3>
    <h4>Hello World H4</h4>
</div>

:::info
You may also use .h1, .h2, .h3, .h4 classes when needed
:::

## Body text

**4 sizes** of body text exist: `lg`, `md` (default), `sm` and `xs`.

**Example**

```jsx
<p className="body-text lg">
    Lorem ipsum, dolor sit amet consectetur adipisicing elit. Rem, accusamus.
</p>

<p className="body-text md">
    Lorem ipsum, dolor sit amet consectetur adipisicing elit. Rem, accusamus.
</p>

<p className="body-text sm">
    Lorem ipsum, dolor sit amet consectetur adipisicing elit. Rem, accusamus.
</p>

<p className="body-text xs">
    Lorem ipsum, dolor sit amet consectetur adipisicing elit. Rem, accusamus.
</p>
```

<div className="bg-blue-light rounded p-20">
    <p className="body-text lg mb-10">
        Lorem ipsum, dolor sit amet consectetur adipisicing elit
    </p>
    <p className="body-text md mb-10">
        Lorem ipsum, dolor sit amet consectetur adipisicing elit
    </p>
    <p className="body-text sm mb-10">
        Lorem ipsum, dolor sit amet consectetur adipisicing elit
    </p>
    <p className="body-text xs mb-10">
        Lorem ipsum, dolor sit amet consectetur adipisicing elit
    </p>
</div>

## Label text

**1 size** of label text exists:`sm`.

:::info
This is a text style for labeling UI content and not specifically for `<label/>` elements
:::

**Example**

```jsx
<span className="label-text sm text-neutral-3">Labeling Text</span>
```

<div className="bg-blue-light rounded p-20">
    <span className="label-text sm text-neutral-3">Labeling Text</span>
</div>

## Button text

**2 sizes** of button text exist: `md` & `sm` (md can be omitted).

**Example**

```jsx
<p className="button-text text-neutral-2">Button Text</p>
<p className="button-text sm text-neutral-2">Button Text</p>
```

<div className="bg-blue-light rounded p-20">
    <p className="button-text text-neutral-2">Regular Button Text</p>
    <p className="button-text sm text-neutral-2">Small Button Text</p>
</div>
