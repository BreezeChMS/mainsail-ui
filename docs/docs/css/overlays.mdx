# Overlays

Using existing position utility classes you can craft an overlay without much css required.

## Overlay

This example contains an `content-overlay` that spans the full ranges of its parent.

<div className="px-15 py-8 mb-20 body-text relative">
    <p>
        Lorem ipsum dolor sit amet consectetur adipisicing elit. Corrupti fugiat
        fugit, harum neque eum dolorem maiores sint mollitia optio dicta!
    </p>
    <div className="content-overlay absolute top-0 right-0 bottom-0 left-0"></div>
</div>

## Blur

This example contains a `content-blur` that blurs all the children.

<div className="px-15 py-8 body-text content-blur">
    <p>
        Lorem ipsum dolor sit amet consectetur adipisicing elit. Corrupti fugiat
        fugit, harum neque eum dolorem maiores sint mollitia optio dicta!
    </p>
</div>
